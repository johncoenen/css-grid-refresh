Grid auto-flow-column

auto-flow default behavior:

  • layout elements by row
  • work along row until no more slots
  • move to next row
  • if row is not declared...an implicit grid track will be created to hold the items.

grid-auto-flow property

  • default value is row but can specify column
  • elements will then layout column by column...
  • ...adding new columns as needed.
